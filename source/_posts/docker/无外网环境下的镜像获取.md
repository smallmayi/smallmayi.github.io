### 一．导出镜像

```shell
# 查看 MySQL 镜像名称和版本
docker images

# 导出镜像为 .tar 文件（例如：mysql:5.7）
docker save -o mysql_image.tar mysql:5.7
```



###　二．传输镜像

```sh
scp mysql_image.tar user@目标机器IP:/目标路径/
```

### 三．目标机器加载镜像

```shell
docker load -i mysql_image.tar
```
